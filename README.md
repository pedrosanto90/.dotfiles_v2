# My dotfiles for Debian with i3

## Instalation
## The install script is made for Debian 13
## Need to have git installed
```bash
sudo apt install git
```
## After install Debian 13 minimal run the following with


```bash
git clone https://github.com/pedrosanto90/.dotfiles_v2.git
cd .dotfiles_v2
sudo ./install.sh
```

# The script will install all the necessary dependecies for all the software
